## Dataset Description

The analysis is based on two main datasets:

Customer Data

Demographics (age group, gender, education, marital status)

Income group, job type, dependents

Credit card category (Silver, Blue, Gold, Platinum)

Transaction Data

Transaction amount & count

Expense type (Bills, Fuel, Grocery, Travel, etc.)

Payment mode (Swipe, Chip, Online)

Interest earned & revenue

Weekly and quarterly timelines
